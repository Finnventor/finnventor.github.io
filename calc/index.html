Help

To use the calculator, simply type in the box provided. No need to press enter, answers will be displayed automatically below the input field. If the input field is underlined in red, the calculator can't understand your query. Try a simpler one.

Examples

2 + 2
4

sin(0.5pi)
1

10 A * 30 V * 1 day in kWh
7.2 kWh

x = sqrt(-18/2)
3i

5x + 4
4 + 15i

Tips

When enter is pressed, the field is cleared but the answer remains. It can also be accessed using the variable ans. Note that if x = 10 is in the input field and the field is cleared using backspace instead of enter, the calculator will see x = 10 and then x = 1, resulting in the variable being set as 1 instead of 10.
